#Issues to add
* HTML does not validate
* missing label in form
* missing for/id for form label
* Image missing an alt attribute (automatable)
* Image that conveys meaning does not contain an alt attribute that communicates the meaning (manual)
    ** example include editorial image
    ** image with text in it (button, navigation)
    ** info graphic
    ** icon
* Font Icon does not contain text
* Font icon does not hide css content (http://www.filamentgroup.com/lab/bulletproof_icon_fonts.html)
* Colour alone is being used to communicate information
* Adjacent image and text links to the same place
* Link text to that not convey information about destination - e.g 'Click here'
* Table used for layout
* Focus not set to modal
* Close button not focusable
* Inifinite scroll with no way to tab out of it
* No focus state on focable items (links, buttons and form items)
